# leet-in-oh

A personal collection of LeetCode problem solutions written in Python. When I hold office hours as a CS teaching assistant but there are no students present and nothing to grade I get in some LeetCode practice, thus leet-in-oh.

## Structure

Solutions are saved as individual `.py` files, typically named after the problem number and stored in the folder form the challenge.

## Languages

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

## Goals

- Build consistency with weekly practice
- Strengthen problem-solving intuition across different topics (arrays, trees, DP, graphs, etc.)
- Keep a personal reference of approaches

## Note

These are my own solutions worked out during practice, not necessarily the most optimal but there for me to revisit and understand.
